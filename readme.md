# 404 Error Page

A responsive error page designed to provide a smooth and user-friendly experience when a page is not found.  
This project focuses on clean design, subtle interactivity, and adaptability across different screen sizes.

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## Project Goal
The goal of this project is to practice front-end development by creating a simple yet effective error page that enhances user experience through responsive design and visual clarity.

## Features
- Responsive layout for all screen sizes  
- Clean and modern UI  
- Lightweight and fast loading  
- Basic interactivity with JavaScript  
- User-friendly error messaging  

## Responsiveness
This project is fully responsive and optimized for:
- Desktop  
- Tablet  
- Mobile devices
